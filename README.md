## вҡҷ Installation

### 1пёҸвғЈ Clone the Repository

```
git clone https://github.com/YOUR_USERNAME/employee-analytics-system.git
```

### 2пёҸвғЈ Navigate to the Project

```
cd employee-analytics-system
```

### 3пёҸвғЈ Create Virtual Environment

```
python -m venv venv
```

### 4пёҸвғЈ Activate Environment

Windows

```
venv\Scripts\activate
```

Mac/Linux

```
source venv/bin/activate
```

### 5пёҸвғЈ Install Dependencies

```
pip install -r requirements.txt
```

### 6пёҸвғЈ Run Migrations

```
python manage.py migrate
```

### 7пёҸвғЈ Start the Server

```
python manage.py runserver
```

Open in browser:

```
http://127.0.0.1:8000
```
